European Portuguese Learning App
About the Project:
I built a web application to help students practice European Portuguese. The app automatically takes vocabulary from class documents (PDFs) and turns them into fun, interactive games to help users learn and remember words.

What the App Does:
The app has 6 different practice sections:
Flashcards: Users can see a Portuguese word, listen to how it sounds, and click to see the English translation and an example sentence.
Vocabulary Quiz: A multiple-choice game where users try to choose the correct English meaning of a Portuguese word.
Fill in the Blanks: A grammar and memory game where users read a Portuguese sentence and have to type the missing word.
Listening Challenge: Users listen to a spoken Portuguese word or phrase and must understand it just by hearing.
Word Pairs: A matching game where users connect the Portuguese word to the correct English translation.
Speed Round: A fast game to test how quickly users can remember words under pressure.

Technologies I Used:
Python: The main programming language I used to write the code.
Streamlit: A tool I used to build the website quickly and make it look professional.
Google Gemini AI: I used Artificial Intelligence to read my class PDFs, find the most important words, and write example sentences automatically.
Pandas: A tool for organizing data. I used it to save the words in a file (CSV) and to make sure the app never saves the same word twice.
gTTS (Google Text-to-Speech): I added this so the app can speak. It lets users hear the correct European Portuguese pronunciation.
GitHub & Streamlit Cloud: I used these to put my app on the internet so anyone can use it for free.
